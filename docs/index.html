電子賀卡寄送器

這是新的流程：點按鈕跳 Google 彈窗授權（scope: gmail.send）。授權後即可直接在頁面寄信，無需再貼 /exec。 如何取得 Client ID？
快速取得 Client ID（一次設定）：
1. 去 Google Cloud Console 新增專案。
2. 啟用「Gmail API」。
3. 建立憑證 → OAuth 用戶端 ID（類型選「網頁應用程式」）。
4. 在「已授權的 JavaScript 來源」加入你的 GitHub Pages 網址（例如 https://你的帳號.github.io）以及 http://localhost（測試用）。
5. 複製 Client ID 貼回上方欄位並保存。
第一次授權時用「測試使用者」即可（不需要審核）。
